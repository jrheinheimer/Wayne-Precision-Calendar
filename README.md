# Wayne Precision Calendar (WPC)

## Leap Year Rules
A year is a leap year if:
1. Divisible by 4
2. Not divisible by 100
3. Except if divisible by 400
4. Except if divisible by 3200

## Average Year Length
365.2421875 days
Drift: ~1 day per 670,000 years

## Description
The WPC is a calendar system more accurate than Gregorian.
